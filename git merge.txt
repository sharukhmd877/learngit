-> fast forward merge
	created a file 
	initialize the git
	commit the changes --> git commit -m "commit 1"
	created a new branch ---> git checkout -b dev
	in dev branch = do some changes in file locally 
	commit the changes in dev branch --->  git commit -am "commit 2 in dev"
										   git log (2 commit can be seen)
	git checkout master
 now i want "commit 2 in dev" to be in my master branch
         git log (only one commit can be seen)
         git merge dev
         git log (can see two commit)
